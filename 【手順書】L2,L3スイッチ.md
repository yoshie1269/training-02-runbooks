# VLAN 作成 手順書（VLAN10 + アクセスポート設定）

## 概要

本手順書では、スイッチ上で VLAN 10 を作成し、指定ポート（FastEthernet0/1）を VLAN 10 のアクセスポートとして設定する。

## 前提

- 対象機器：L2 スイッチ（Packet Tracer 等を想定）
- 設定対象ポート：FastEthernet0/1
- VLAN ID：10


## 1. VLAN を作成する


### この工程でしていること
- スイッチに VLAN 10 を「論理的なネットワーク（ブロードキャストドメイン）」として追加する


### 実行コマンド
```bash
enable
conf t

vlan 10
exit
```

### コマンドの意味

| コマンド | 意味 |
|---|---|
| enable | 特権モードへ移行 |
| conf t | グローバル設定モード |
| vlan 10 | VLAN ID 10 を作成（VLAN設定モードへ） |
| exit | 1つ前のモードへ戻る |


### 確認方法とOKの目安
```bash
show vlan brief
```
- VLAN 一覧に `10` が表示される  
- Status が `active` になっている


## 2. アクセスポートを VLAN 10 に所属させる（Fa0/1）


### この工程でしていること
- FastEthernet0/1 を「1つの VLAN のみ所属するポート（Access）」として設定する
- VLAN 10 の端末を接続するポートとして利用できる状態にする


### 実行コマンド
```bash
interface fastethernet 0/1
switchport mode access
switchport access vlan 10
```


### コマンドの意味

| コマンド | 意味 |
|---|---|
| interface fa0/1 | 設定対象ポートを選択 |
| switchport mode access | アクセスポート化（トランクではなく端末用） |
| switchport access vlan 10 | VLAN 10 に所属させる |


### 確認方法とOKの目安

#### VLAN 観点で確認
```bash
show vlan brief
```
- VLAN10 の Ports 欄に `Fa0/1` が表示される

#### インターフェース設定で確認
```bash
show running-config interface fastethernet 0/1
```
- 以下が含まれること
```bash
switchport mode access
switchport access vlan 10
```

## よくあるミスと注意点

- `switchport access vlan 10` の vlan を plan と書くと設定できない（コマンド不一致）
- `switchport mode access` を入れないと、環境によっては期待通り VLAN に入らないことがある
- VLAN を作ってもポートを所属させないと意味がない（端末が VLAN10 に入らない）



## 完了条件

- VLAN 10 が作成されている
- Fa0/1 が VLAN 10 のアクセスポートになっている
